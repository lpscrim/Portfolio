Building Meaningful Digital Spaces

I'm a self-taught Front-end developer and designer dedicated to crafting digital experiences that are both functional and visually compelling. I believe that if we must spend so much time in digital spaces, they should atleast be intuitive and engaging.


Transforming ideas into interactive digital spaces, I aim to create websites that are responsive, user-friendly, and visually appealing. My aim is to to help businesses and individuals communicate their stories effectively through thoughtful, well-designed web interfaces.

Web Projects

Over the course of my career I have built a variety of websites that explore different requirements and technical approaches. Here are some of them:

Cioch Outdoors

An overhaul of an existing website for a local outdoor clothing manufacturer, built using brand colors and style.

Using SEO and responsive design principles the new site was created for use with modern devices and have a streamlined interface. The brands charm and personality was also a key focus of the redesign.

Lampman

This E-commerce website was built using NextJS and Tailwind CSS for an active antique business, specialising in bespoke one off lamps.

With no need for a full IMS I created a stock management solution using only Stripe API to take secure payments and remove bought items from the inventory.

Portfolio

After using various frameworks for serverside projects and API requests, I decided to return to HTML/vanillaJS to create a simple and fast postfolio website.

As a result the simple website has achieved 100% scores on lighthouse, is lightning fast and extremely complient. Browsers that do not support Javascript will still be able to view the full site.
